How to write a (basic) compiler/interpreter
In this course, I'll cover the basics of how to write your own compiler and/or interpreter. Theoretically, you could write your own python interpreter after finishing this course, even if it probably wouldn't be as fast as the official ones.
We won't be covering such things as optimization (such as loop unrolling, for example), but we'll cover the basics of how you could find such patterns. I'll leave the rest up to you. The course is split up into four lessons, which are listed below. Click on where you'd like to start from (I'd recommend lesson 0 or 1), and go from there.
Note that you can move between lessons using the left and right arrows on the header, once on a lesson page.
