How to set up your working environment
forward
Since you're reading this, I'll assume you have access to a browserIf you don't, and are reading this via wget or some other insane way of doing things, God help you, because I sure won't.. I'll also assume that you have some basic knowledge of programming, since this should NOT be your first foray into the subject. So, somewhere on your computer, save the following HTML file: <!DOCTYPE html> <html> <head> <script src="index.js"></script> </head> <body> <textarea id="code"></textarea><br> <button id="btn">Compile</code> </body> </html> Then, in the same folder, save the following JS file as "index.js": window.addEventListener('DOMContentLoaded', () => { document.getElementById('btn').addEventListener('click', () => compile(document.getElementById('code').value)) }); function compile(code) { // Write your code here } Open the HTML file in your browser (dragging it into an empty tab should work), and then you're basically done. Write/paste the code you want to test into the field, then press the "compile" button.
Open up the browser console (you'll have to look up how to do this for your browser yourself) to see logging, errors, results, whatever.
Remember to refresh the page between code changes!
