Introduction

This course will follow the construction of two languages:
  • a calculator language:
    3 * (2 + 5)
  • a scripting language:
    OUTPUT "What's your name? "; var name = INPUT; OUTPUT "Hello, " + name + "!\n";
Both languages will be interpreted, and written in JavaScript. I'll leave it to you to write a compiled language, but you should understand how to do it after having taking this course.
The scripting language is meant for people who are interested in getting to know things on a deeper level. We'll go into some more advanced concepts (variables and functions, for example). Each lesson, you will find sections like these:
