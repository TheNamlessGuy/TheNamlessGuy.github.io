I started programming in school at about 17, and I haven't stopped since (except for when I play games).
My favorite programming languages are C/C++. I list both at the same time, since I tend to prefer writing C code, but sometimes using the C++ standard library. I tend to prefer languages where I can myself determine exactly what is happening, which is very easy in C/C++ (except maybe in modern C++, but that's a whole different issue). The best example I have is the fact that it is always the programmer who determines if a value is pass by reference or pass by value in those languages, while in for example Java, all objects are always pass by reference (unless you do some really obscure voodoo).
