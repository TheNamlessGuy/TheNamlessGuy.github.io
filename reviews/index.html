Search help
To search, just write your query into the search box. Each word (separated by space) is searched for in the title and description of the review.
If you want to search a specific field, prefix the word with:
t: or tag: to search tags (search word must match exactly)
score: to search the score (search word must match exactly)
title: to search the title
desc: to search the description
date: to search the reviewed at date (result must start with search word. date:2021 for all reviews from 2021, for example)
For example, to search for games, type t:game.
You can prefix a search word with - in order to negate the search. If it matches a review, the review won't show up. For example, no games: -t:game
If all search words match, it shows up!
